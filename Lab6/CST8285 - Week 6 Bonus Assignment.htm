CST8285 Bonus Assignment

This assignment is not mandatory, however I highly suggest that you attempt this assignment. The concepts used in this assignment will be used in later labs, so this is a good opportunity to try them out.

The purpose of this assignment is to use JavaScript to validate the fields in a form prior to the form being submitted. I highly recommend that you watch section 3 of the lynda.com course called Validating and Processing Forms with JavaScript and PHP prior to attempting this assignment. Also, Chapter 6 from the textbook provides an in depth look at how to use JavaScript to interact with forms.

There will be a lecture on this material after the midterm.

Instructions

1. Download the file Week6Bonus.zip from BlackBoard

Please download and extract the Week6Bonus.zip file to the hard drive on your computer. Inside you will find a copy of the solution for Week 4's Lab (the registration form).

2. Add a <script> element to the <head> element

This is where you will put your form validation code

3. Create a function called validateForm().

4. Using JavaScript, add code the the validateForm() function to validate the form values.

Write some code that will validate the form is submitted, and inform the user about what needs to be changed. You are free to choose how you want to inform the user. You may use an alert, or a hidden <div beside each form field that appears when there are errors, or any way you desire.

Your code should check the following on the form:

  1. Verify that all of the required text boxes have some text in them.
  2. Verify that the user has picked a province in the dropdown box.
  3. Verify that the value in the credit card number is actually a number, and that it is greater than or equal to 15 characters. If you are feeling really adventurous, you can verify that if they select Visa or MasterCard, that the length of the credit card number is 16 digits, but if they select AMEX, that the length is 15 characters.
  4. Verify that the 'Agree to terms' checkbox has been checked.

5. Upload the project to a folder called Week6Bonus in your webspace.

Create a folder called Week6Bonus in your webspace. Upload all of the assignment files, including the images and css folders.

6. Blackboard Submission

Submit your webspace's Week6Bonus.html URL to Blackboard's dropbox. TEST TO MAKE SURE YOUR URL WORKS AND THE PAGE DISPLAYS CORRECTLY.
